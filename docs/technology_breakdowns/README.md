# Разборы технологий проекта

Эта папка хранит технические разборы подсистем дипломного проекта
по поисково-спасательному БПЛА.

Планируемые темы:

- захват и сопровождение цели;
- миссионный конечный автомат;
- YOLO12s и обработка изображения;
- PX4 Offboard управление;
- построение карты сцены;
- A* планирование эвакуационного маршрута;
- интеграция ROS 2, PX4 SITL и Gazebo GZ.

Готовые разборы:

- [Захват и наведение на цель](target_acquisition.md)
- [Миссионный конечный автомат](mission_state_machine.md)
